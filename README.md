# 🎮 Tic Tac Toe Game

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript (ES6).

## 📸 Previews
![Game UI](./screenshots/home.png)

![Play Game UI](./screenshots/play-preview.png)

## 🔗 Live Demo
[🎮 Play Live Game]( https://kg-se.github.io/tic-tac-toe_game/)

## 🚀 Features
- Start/close game functionality
- Two-player mode
- Dynamic player names
- Winner detection
- Draw handling
- Clean UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (ES6)
